# bbcode
bbcode parser Bundle for Symfony framework


## works:
- Bold [b]..[/b] *
- Italic [i]..[/i] *
- Underline ([u]..[/u] - span style
- Colors [color=(black/#cccccc)]..[/color] - span style

## will work soon:
- img
- links a with targets
- video (youtube?)
- ...

(*) probable will be changed to span
